WELL, WHAT THE HELL IS THIS?

This is a customized Android Kernel. Currently this is only available for Realme 3 Pro (RMX1851), but we have plans to expand it to other devices as well.

OK, SO THIS IS A KERNEL. WHAT ARE ITS "FEATURES"?

  • Linux Upstreams
  • LZ4 compression for zram
  • FSync Switch
  • CRC Switch
  • Klapse
  • Westwood TCP
  • BFQ IO Scheduler
  • Vibration Control
  • Sound Control
  • Built using Link Time Optimizations (LTO)
  • Built with Clang 12

So, the features are amazing, but does it support my ROM?

This kernel supports any ROM (Android 10/11) based on RealmeUI (RUI) and RUI itself. This also supports the OOS GSI ported by Zaxceed. ( Yeah enjoy, you Oxygen lovers!!!)

Yay, my ROM supports this!!! But how do I flash this?

  • First flash/restore stock kernel ( Only if you are coming from some custom kernel)
  • Flash magisk
  • Flash kernel zip

If magisk is already installed, then no need to reinstall it. The kernel zip will automatically detect magisk and retain it.
